# Plugins Overview

Official plugins for common web framework needs.

| Plugin | Description |
|---|---|
| ziez-cors | CORS middleware with origin whitelist/predicate |
| ziez-compression | gzip/deflate/brotli response compression |
| ziez-security | Helmet + XSS protection middleware |
| ziez-static | Static file serving middleware |
| ziez-template | Template engine with layouts and caching |
| ziez-tls | TLS/HTTPS with HTTP→HTTPS redirect |
| ziez-tracker | Request logging with UA parsing |
| ziez-ua-parser | User-Agent parser (standalone) |
